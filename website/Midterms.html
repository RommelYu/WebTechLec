    • INTRODUCTION TO CSS (CASCADING STYLE SHEET)

    The Cascading Style Sheet (CSS) is a language used to specify the presentation (e.g. Layout, Formatting, Fonts, Colors, etc) of structurally marked up documents (e.g. HTML, XML, SVG, MathML, etc).

    Cascasing Style Sheet (CSS) was developed by Hakon Wium Lie (CHSS – Cascading HTML Style Sheet) and Bert Bos (SSP).

    DATE VERSION HISTORY
    December 1996 CSS Level 1 (CSS, W3C Recommendation)
    May 1998 CSS Level 2 (CSS, W3C Recommendation)
    June 2011 CSS Level 3 Revision 1 (CSS, W3C Recommendation)
    - CSS Level 3 – Modular approach to CSS development (as opposed to the monolithic specification of CSS 2.1)

    ABOUT CSS


    TYPE DESCRIPTION
    CSS Rule Set Consists of a selector, followed by a brace-enclosed declaration block, which contain zero or more semi-colon (;) separated property declarations which in term consists of a property-name, followed by a colon (:), followed by a property name.
    CSS Selectors Structured used as a condition in a CSS rule to determine with element in the document tree and matched by the selector and are thus targeted by the formatting.
    CSS Syntax Chance of one or more sequences of simple selectors separated by combinators.
    Starts with type selector a universaseparated (may be implied).
    Cannot contain other type.
    CSS Statement
    @charset
    - Defines the character set used by the style sheet.
    @media
    - A conditional group rule that will apply its content if the device meets the criteria of the condition defined using a media query.
    @supports
    - A conditional group rule that will apply its content if the browser meets the criteria of the given condition.
    @import
    - Tells the CSS engine to include an external style sheet.
    @namespace
    - Tells the CSS engine that all its content must be considered prefixed with an XML namespace.
    @page
    - Describes the aspect of layout changes that will be applied when printing the document.
    @font-face
    - Describes the aspect of an external font to be downloaded.
    @keyframe
    - Describes the aspect of intermediate steps in a CSS animation sequence.
    Selector Group Comma separated list of selectors representing the union of all elements selected by each of h selector in the list.

    Child Combinator (>)
    Sibling Combinator - Adjacent (+), General (-)

    :first-line, ::first-line
    :before, ::before
