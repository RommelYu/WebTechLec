HTML - Hypertext Markup Language

  • Html was used to specify the structure and content of the web pages as well as their presentation.
  • Modern web development uses html for sematic specification of web pages structure and content with presentational aspects being delegted to styesheet
  • Html initially developed at CERN, then EITF, W3C and also WHATWG

Version History

  • Html tags, by Tim Berners-Lee late 1991
  • Html drafted by Tim Berners-Lee and Dan Connolly
  • HTML 3.2 (Jan 1997)
  • HTML 4.0 (Dec 1997)
  • HTML 4.01 (Dec 1999)
